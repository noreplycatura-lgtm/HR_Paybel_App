# **App Name**: Novita HR Portal

## Core Features:

- Login Portal: Secure login portal with username/password authentication.
- Attendance Management: Attendance upload via Excel sheet with predefined columns and color-coded attendance values (P, A, HD, PH, SL, CL, PL, W).
- Leave Management: Display automatically calculated monthly leave credits (CL, SL, PL) and balances. PL carry forward to the next year, while CL and SL reset.
- Salary Preparation & Management: Two methods for salary data entry: Excel upload and manual form entry. Display Basic Salary and calculated monthly components (HRA, CA, Medical, Other Allowance) for the Final Gross Salary.
- Salary Slip Generator: Auto-generate monthly PDF salary slips with employee details, attendance summary, leave balance, earnings, deductions, and net salary.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to convey trustworthiness and stability, aligning with HR's responsibilities.
- Background color: Light grayish-blue (#D3E8F2), very close in hue to the primary but desaturated, to create a calm, professional backdrop.
- Accent color: Violet (#8A2BE2) for interactive elements, to stand out against the primary color and create visual interest.
- Clean, sans-serif fonts for readability and a professional appearance.
- Consistent use of outline-style icons for a modern, uncluttered look.
- Well-spaced layout with clear section divisions for easy navigation.
